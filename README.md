# FocusMind
FocusMind es una aplicación web de gestión del tiempo basada en la Técnica Pomodoro . Está diseñado para ayudar a los usuarios a maximizar su concentración y prevenir el agotamiento mediante ciclos de trabajo y descanso estructurados.
Características principales:

Ciclos dinámicos: Alterna automáticamente entre períodos de trabajo (25 min) y descansos (5 min) con cambios visuales de color (Rojo para enfocar, Verde para relajarse).

Personalización Total: Permite al usuario configurar tiempos a medida según sus necesidades de flujo de trabajo.

Sistema de Alertas: Notificaciones sonoras integradas con opción de silenciado rápido.

Historial de Actividad: Registro en tiempo real de las sesiones completadas para realizar un seguimiento del progreso diario.

Diseño Adaptable: Interfaz limpia y moderna construida con tecnologías estándar (HTML5, CSS3 y JavaScript Vanilla) que funciona en cualquier navegador.
